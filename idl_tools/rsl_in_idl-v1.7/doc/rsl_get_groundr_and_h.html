rsl_get_groundr_and_h

Get ground range and height given slant range and elevation.

Syntax

rsl_get_groundr_and_h, slant_range, elevation, ground_range, height

Arguments

Input
slant_range Range to radar target, in kilometers.
elevation Elevation angle in degrees.
Output
ground_range Ground range to radar target, in kilometers.
height Height in kilometers.

Keywords

None
